## Mengenalgebra-Gesetze

| Gesetz                                             | Name                      |
|----------------------------------------------------|---------------------------|
| X ∪ Y = Y ∪ X                                      | Kommutativgesetze         |
| X ∩ Y = Y ∩ X                                      | Kommutativgesetze         |
| (X ∪ Y) ∪ Z = X ∪ (Y ∪ Z)                          | Assoziativgesetze         |
| (X ∩ Y) ∩ Z = X ∩ (Y ∩ Z)                          | Assoziativgesetze         |
| (X ∪ Y) ∩ Z = (X ∩ Z) ∪ (Y ∩ Z)                    | Distributivgesetze        |
| (X ∩ Y) ∪ Z = (X ∪ Z) ∩ (Y ∪ Z)                    | Distributivgesetze        |
| X ∪ (X ∩ Y) = X                                    | Absorptionsgesetze        |
| X ∩ (X ∪ Y) = X                                    | Absorptionsgesetze        |
| ¬(X ∪ Y) = ¬X ∩ ¬Y                                 | Gesetze von de Morgan     |
| ¬(X ∩ Y) = ¬X ∪ ¬Y                                 | Gesetze von de Morgan     |
